# 庐山派 K230 引脚信息整理（基于 CSDN 文章）

## 文档说明
- 来源文章：<https://blog.csdn.net/weixin_46151516/article/details/144896532>
- 原文标题：`庐山派K230学习日记3 外设模块GPIO`
- 整理目标：提取和汇总开发中最常用的引脚与复用信息，便于快速查阅。

## 1. 开发板常用引脚速查

### 1.1 40Pin 排针中与 I2C 相关的实际 GPIO
| 接口功能 | 实际引脚 |
| --- | --- |
| I2C0_SCL | BANK3_GPIO48 |
| I2C0_SDA | BANK3_GPIO49 |
| I2C1_SCL | BANK3_GPIO40 |
| I2C1_SDA | BANK3_GPIO41 |

补充说明：
- 文中说明 I2C0/I2C1 兼顾摄像头驱动场景，已有内部上拉，外扩时通常不必重复加上拉。

### 1.2 GH1.25 与焊盘引出的串口引脚
| 串口 | TX | RX | 备注 |
| --- | --- | --- | --- |
| UART0 | BANK3_GPIO38 | BANK3_GPIO39 | 默认被 RT-Smart 占用作调试口 |
| UART2 | BANK0_GPIO11 | BANK0_GPIO12 | CanMV 常用可用串口 |
| UART3 | BANK4_GPIO50 | BANK4_GPIO51 | CanMV 常用可用串口 |

### 1.3 板载 RGB 灯引脚
| 颜色 | GPIO |
| --- | --- |
| 红灯 R | GPIO62 |
| 绿灯 G | GPIO20 |
| 蓝灯 B | GPIO63 |

补充说明：
- 板载 RGB 为共阳结构，`低电平点亮，高电平熄灭`。

### 1.4 ADC 相关注意事项
- 40Pin 排针不提供 ADC 功能。
- K230 ADC 最高输入电压为 1.8V。
- ADC 引脚仅可作模拟输入，不可作普通 GPIO。

### 1.5 K230Vision 对接 ESP32（当前工程可运行口径）
| 方向 | K230 端 | ESP32 端 |
| --- | --- | --- |
| K230 -> ESP32 | UART1_TX | GPIO18 (RX) |
| ESP32 -> K230 | UART1_RX | GPIO17 (TX) |
| 共地 | GND | GND |

补充说明：
- 当前项目代码使用 `APP_CONFIG["uart_id"] = 1`（即 UART1）。
- 引脚定义以 K230 工程文档与本文件口径为准。
- 本项目当前未在代码中显式调用 `FPIOA.set_function(...)`，按现有固件映射直接对接。
- `enable_esp32_uart` 默认值由项目阶段控制，联调时按需手动开启。

## 3. 使用建议（结合 K230Vision 项目）
- 如需与 ESP32 串口通信，优先选用 `UART2(GPIO11/12)` 或 `UART3(GPIO50/51)`，避免占用默认调试口 UART0。
- 若使用板载 RGB 做状态灯，可直接使用 `GPIO62/20/63`，并按共阳逻辑控制电平。
- 涉及 I2C 外设时，优先核对实际映射到 `GPIO48/49` 或 `GPIO40/41`，避免仅按接口名接线。
- 引脚最终复用状态建议在板端运行 `FPIOA().help()` 再次确认，防止被其他模块重映射。

## 4. 附：原文中的关键 API（便于复查）
- `fpioa.set_function(pin, func, ...)`：设置引脚复用功能。
- `fpioa.get_pin_func(pin)`：查询某引脚当前功能号。
- `fpioa.get_pin_num(func)`：查询某功能当前绑定到哪个引脚。
- `fpioa.help()`：打印全部引脚复用信息。
