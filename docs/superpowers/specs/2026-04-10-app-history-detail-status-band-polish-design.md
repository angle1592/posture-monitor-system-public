# App 当天明细状态时间带精修设计

## 1. 背景

当天明细已经从折线改成了状态时间带，方向正确，但当前版本仍有明显视觉问题：

1. 底部时间标签出现重复
2. 状态带过粗，像进度条而不是时间线
3. 异常段与正常段区分不够清楚

## 2. 目标

1. 保留“当天明细 = 状态时间带”的方向。
2. 让时间带更像精细时间轨道，而不是粗色条。
3. 只保留一排时间标签。
4. 明确区分：正常 / 异常 / 无人。

## 3. 方案

### 3.1 轨道形态

- 使用一条较细的底部轨道作为时间基线
- 正常段与异常段绘制为圆角线段
- 无人段保持留空断开

### 3.2 视觉层级

- 正常：绿色细轨道
- 异常：橙色高亮细轨道
- 混合：绿橙渐变细轨道
- 背景轨道：低对比度灰蓝色

### 3.3 辅助信息

- 删除 canvas 内重复时间文字
- 仅保留 HTML 轴标签一排
- 在提示文案下方增加简洁图例

## 4. 影响范围

- `app/src/pages/history/index.vue`

不改数据层与评分层。
